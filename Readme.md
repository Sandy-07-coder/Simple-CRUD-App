# Simple CRUD App 🛠️

I have built a backend server using **Node.js**, **Express.js**, and **MongoDB**.  
The project follows a clear file structure with **server**, **controllers**, and **routes** folders.  

This is one of my first backend projects, built during my weekends. I consider it as my **initial step into backend development**, coming from a frontend background.  

---

## 🚀 Tech Stack
- **Node.js**
- **Express.js**
- **MongoDB**
